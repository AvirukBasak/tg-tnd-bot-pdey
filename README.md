# Telegram Bot: Truth n Dare
- Date: Aug 7, 2022
- Client: Pammy Dey
- Platform: NodeJS

## Bot UI
```
/help  : send help message
/start : same as help, with some more info
/spin  : randomly chooses 2 participants, if < 2, appropriate message is sent
/stop  : game ends
```
